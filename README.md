# (C# 코딩) 에코 메신저

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 사용자의 키보드 입력을 받아 처리하는 프로그램
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, Github
- 사용한 컨트롤 :
  - TextBox, Button, Label, ListBox.
- 사용한 기술과 구현한 기능:
  - 이벤트 : Button 클릭 이벤트를 처리하여 사용자의 입력을 받아 ListBox에 추가하는 기능 구현
  - UI 디자인: Windows Forms를 사용하여 간단한 사용자 인터페이스 구성
  - 데이터 관리: ListBox를 활용하여 입력된 메시지를 저장하고 표시하는 기능 구현
	- 


## 실행 화면 (과제1)
- 과제 1 코드의 실행 스크린샷

! [과제1 실행화면] (img/과제1 실행.png)

- 과제 내용
  - Label, TextBox, Button 컨트롤을 사용하여 간단한 메신저 UI를 구성합니다.
  - '전송' 버튼 클릭 시 TextBox에 입력된 텍스트를 ListBox에 추가하는 기능 구현합니다.
  - 추가 직후 TextBox는 초기화되어 다음 입력을 받을 준비가 되도록 설정합니다.

- 구현 내용과 기능 설명
  - Label: "메시지 입력" 텍스트를 표시하여 사용자가 어디에 메시지를 입력해야 하는지 안내합니다.
  - TextBox: 사용자가 메시지를 입력할 수 있는 공간입니다. 사용자가 텍스트를 입력한 후 '전송' 버튼을 클릭하면 이 텍스트가 ListBox에 추가됩니다.
  - Button: "전송" 버튼을 클릭하면 TextBox에 입력된 텍스트가 ListBox에 추가되고, TextBox는 초기화되어 다음 메시지를 입력할 준비가 됩니다.
  - ListBox: 사용자가 입력한 메시지들이 순서대로 저장되고 표시되는 공간입니다. 새로운 메시지가 추가될 때마다 ListBox에 업데이트됩니다.
	


## 실행 화면 (과제2)
- 과제 2 코드의 실행 스크린샷

![과제2 실행화면] (img/과제2 실행.png)

- 과제 내용
 - 과제 1에서 구현한 기능에 추가하여, 사용자가 입력한 메시지를 ListBox에 저장하는 기능을 구현합니다.
 - 전송 후에 마우스로 입력창을 다시 클릭하지 않아도 되도록 커서를 자동으로 입력창에 둡니다.
 - 마우스 클릭 대신 키보드의 Enter 키를 눌러도 메시지가 전송되도록 합니다.

- 구현 내용과 기능 설명
  - TextBox의 KeyDown 이벤트를 활용하여 사용자가 Enter 키를 눌렀을 때 메시지가 전송되도록 구현합니다. 
	-> 이를 통해 사용자는 마우스 클릭 없이도 빠르게 메시지를 입력하고 전송할 수 있습니다.
  - 전송 버튼 클릭 시 TextBox에 입력된 텍스트가 ListBox에 추가되고, TextBox는 초기화되어 다음 메시지를 입력할 준비가 됩니다. 
  - 또한, 커서가 자동으로 TextBox에 위치하여 사용자가 바로 다음 메시지를 입력할 수 있도록 합니다.
	

## 실행 화면 (과제3)
- 과제 3 코드의 실행 스크린샷

![과제3 실행화면] (img/과제3 실행.png)

- 과제 내용
 - 타임스탬프를 추가하였습니다. 메시지 앞에 현재시간을 자동으로 메세지 앞에 결합하여 리스트에 출력합니다.
 - 현재 리스트에 쌓인 총 메시지 개수를 계산하여 Label에 실시간으로 업데이트합니다.
 - 사용자가 입력한 메시지의 앞뒤 불필요한 공백을 Trim()함수로 제거하여 저장합니다.

- 구현 내용과 기능 설명
 - srting을 사용하여 [HH:mm:ss] 시간 정보 구하였습니다.
 - label2 Text 사용하여 현재 대화 실시간 업데이트 EchoWindow.Items.Count 사용.
 - Trim 함수를 통해 공백이 발생하는 사용자의 메시지 공백을 제거합니다.
